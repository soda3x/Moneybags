# Moneybags 💰

Moneybags is an open-source cashflow management system used for creating tax invoices. Most of the solutions on the market today cost exorbitant amounts of money or are obtained via silly subscription models. Let's change that :3

## Electron-Rewrite branch

This is the electron re-write branch of Moneybags. The future of Moneybags will be as an electron based application promoting cross-platform compatibility and future proofing as Moneybags was originally developed with the ever ageing WinForms.

The Electron-rewrite branch of Moneybags uses Electron with TypeScript.

### Running Moneybags electron-rewrite

```sh
npm i # Download all the packages
npm start # This will build the TS files and Run Moneybags electron-rewrite
```
